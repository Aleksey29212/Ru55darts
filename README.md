
# DartBrig Pro - Система управления рейтингами (v2.8 Stable)

Современная платформа для автоматизации дартс-турниров. Построена на стеке **Next.js 15**, **Tailwind CSS** и **Firebase**.

## 🛠 Технический стек
- **Framework**: Next.js 15 (App Router)
- **Backend**: Firebase Firestore (Облачная БД)
- **Auth**: Firebase Auth (Anonymous)
- **Deployment**: Docker / Firebase App Hosting

## 🚀 Деплой (Отправка кода)

Для успешной отправки изменений в GitHub, выполните следующие шаги:

1. **Привяжите ваш токен (Один раз)**. Скопируйте и выполните эту команду в терминале, вставив свой Personal Access Token:
   ```bash
   git remote set-url origin https://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@github.com/Aleksey29212/Ru55darts.git
   ```

2. **Запустите деплой**:
   ```bash
   npm run deploy:github
   ```

**Важно по безопасности:**
Скрипт `deploy.sh` больше не содержит вшитых токенов для защиты от блокировок GitHub (Push Protection). Используйте привязку токена через URL репозитория (как показано выше).
