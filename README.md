# DartBrig Pro - Система управления рейтингами (v2.8 Stable)

Современная платформа для автоматизации дартс-турниров. Построена на стеке **Next.js 15**, **Tailwind CSS** и **Firebase**.

## 🛠 Технический стек
- **Framework**: Next.js 15 (App Router)
- **Backend**: Firebase Firestore (Облачная БД)
- **Auth**: Firebase Auth (Anonymous)
- **Output**: `standalone` (Оптимизировано для Timeweb Cloud / Vercel)

## 🚀 Инструкция по деплою (Timeweb Cloud)

### Шаг 1: Отправка в GitHub
1. Откройте терминал в среде разработки.
2. Выполните команду: `npm run deploy:github`
3. Весь код со всеми исправлениями (математика, UI, мобильная верстка) будет отправлен в ваш репозиторий.

### Шаг 2: Настройка в Timeweb Cloud
1. Создайте проект «Node.js приложение».
2. Подключите ваш GitHub репозиторий `Aleksey29212/Ru55darts.git`.
3. В разделе **«Переменные окружения»** добавьте ключи из вашего проекта Firebase:
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=ваш_ключ
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=ваш_домен
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=ваш_id_проекта
   NEXT_PUBLIC_FIREBASE_APP_ID=ваш_id_приложения
   NEXT_PUBLIC_ADMIN_PASSWORD=1234 (или ваш пароль)
   ```
4. Параметры сборки:
   - Команда сборки: `npm run build`
   - Команда запуска: `npm start`
   - Порт: `3000`

## 📸 Хранение данных
- **Турниры**: Сохраняются в Firestore навсегда. Повторный парсинг не требуется.
- **Фотографии**: Загружаются через «Фотостудию», конвертируются в **Base64** и хранятся прямо в БД. Система полностью автономна.
