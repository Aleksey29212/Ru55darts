## 🚀 Быстрый запуск и деплой

### 1. Подготовка и отправка на GitHub
Проект настроен для быстрой отправки. Используйте встроенную команду:
```bash
npm run deploy:github
```
*Этот скрипт инициализирует Git, создаст коммит и отправит код в ваш репозиторий Ru55darts.*

### 2. Настройка окружения (Environment Variables)
В панели управления вашим хостингом (Vercel или Firebase App Hosting) добавьте:
- `NEXT_PUBLIC_ADMIN_PASSWORD`: Ваш уникальный пароль для входа в `/admin`.

### 3. Firebase App Hosting (Рекомендуется)
Приложение оптимизировано под **Firebase App Hosting**. Система автоматически распознает SSR-компоненты и настроит серверную часть при каждом пуше в ветку `main`.

## 💻 Разработка
- `npm run dev` - запуск локального сервера (порт 9002).
- `npm run build` - проверка проекта на ошибки сборки Next.js.
- `npm run lint` - проверка качества кода.
