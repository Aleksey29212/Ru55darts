# DartBrig Pro - Система управления рейтингами (v2.8 Stable)

Современная платформа для автоматизации дартс-турниров и управления профессиональными профилями игроков.

## ✨ Новые возможности (v2.8)
- **10 Мастер-шаблонов**: Полная кастомизация карточек игроков (Киберпанк, Элитный, Ретро, Арена и др.).
- **Интеллектуальные подсказки**: Профессиональная аналитика под знаком «!», адаптированная для смартфонов.
- **Матовое стекло (Deep Blur)**: Фиксированные панели полностью скрывают контент при прокрутке.
- **Умные цифры**: Единый масштаб шрифта (по эталону AVG) с авто-адаптацией под длину данных.
- **Advanced Sponsorship**: Поддержка до 3-х спонсоров с механикой раскрытия промокодов.

---

## 🖥️ Требования к хостингу

Для стабильной работы системы ваш сервер или облачный провайдер должен соответствовать следующим параметрам:

- **Node.js**: 18.17.0 или выше (Рекомендуется v20).
- **Оперативная память**: минимум 512 MB (рекомендуется 1 GB для сборки).
- **База данных**: Google Cloud Firestore (через Firebase SDK).
- **SSL**: Рекомендуется (для работы камеры в фотостудии).

---

## 🐳 Использование Docker (Опционально)

Docker **не обязателен**. Вы можете запустить проект напрямую через Node.js. 
Если вы хотите использовать Docker:
1. Соберите образ: `docker build -t dartbrig-pro .`
2. Запустите контейнер: `docker run -p 3000:3000 dartbrig-pro`

---

## ⚙️ Настройка переменных окружения

Создайте файл `.env.local` в корне проекта или добавьте эти переменные в панель управления вашего хостинга:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=ваш_ключ
NEXT_PUBLIC_FIREBASE_PROJECT_ID=ваш_id_проекта
NEXT_PUBLIC_FIREBASE_APP_ID=ваш_id_приложения
NEXT_PUBLIC_ADMIN_PASSWORD=ваш_секретный_пароль
```

---

## 🧪 Тестирование перед деплоем

Чтобы убедиться, что сайт готов к запуску, выполните эти команды в терминале:

1. **Проверка сборки**:
   ```bash
   npm run build
   ```
   *Если сборка прошла успешно, проект готов к деплою.*

2. **Локальный запуск (Production режим)**:
   ```bash
   npm run start
   ```

## 🚀 Как отправить проект в GitHub

Запустите команду в терминале Firebase Studio для автоматической загрузки:
```bash
npm run deploy:github
```
