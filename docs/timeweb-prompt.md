- Среда выполнения: Node.js 20.
- Режим сборки: Standalone (оптимизированный Docker-образ).

Приложение должно обеспечивать крайне быстрый отклик интерфейса (Snappy UI). Ожидаются пиковые нагрузки во время проведения турниров (одновременные обращения к рейтингам и импорт данных).
