# DartBrig Pro - Система управления рейтингами (v2.6)

Современная платформа для автоматизации дартс-турниров, агрегации статистики и создания премиальных карточек игроков.

## 🚀 Быстрый запуск на GitHub

Чтобы отправить этот проект в свой репозиторий на GitHub, выполните одну команду в терминале:

```bash
npm run deploy:github
```

### Что делает эта команда:
1. Исправляет ошибки в локальной базе Git (если они возникли).
2. Индексирует все файлы и создает коммит.
3. При необходимости предлагает выполнить "Force Push", если история вашего репозитория в GitHub отличается от локальной.

## 🛡️ Безопасность и настройки (Environment Variables)

**ВАЖНО:** Не забудьте настроить переменные окружения на вашем хостинге (Vercel, Firebase App Hosting и др.). См. файл `.env.example` для примера.

| Переменная | Описание | Значение по умолчанию |
| :--- | :--- | :--- |
| `NEXT_PUBLIC_ADMIN_PASSWORD` | Пароль для входа в `/admin` | `1234` |
| `NEXT_PUBLIC_FIREBASE_API_KEY` | API ключ Firebase | — |
| `NEXT_PUBLIC_FIREBASE_PROJECT_ID` | ID проекта Firebase | — |

---

## 🛠️ Основные функции
- **Импорт турниров**: Автоматический парсинг с dartsbase.ru по ID.
- **Лиги как страницы**: Каждая лига (Глобальная, Омск и др.) загружается по требованию.
- **Личные карточки**: Подробная статистика с 10+ параметрами и пояснениями.
- **Студия стилей**: Генерация тем оформления через ИИ (Gemini).
