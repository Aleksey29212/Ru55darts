# DartBrig Pro - Система управления рейтингами (v2.6)

Современная платформа для автоматизации дартс-турниров. Разработано для стабильной работы на любом хостинге.

## 🚀 Как запустить (Инструкция для новичков)

Если вы используете **Timeweb Cloud (VPS)**:

1. **Отправьте код**: Запустите в терминале Firebase Studio команду:
   ```bash
   npm run deploy:github
   ```
2. **Зайдите на сервер**: Подключитесь к вашему серверу через консоль Timeweb или SSH.
3. **Настройте ключи**: В корневой папке проекта создайте файл `.env.local` и вставьте туда данные из Firebase Console.
4. **Запуск**:
   ```bash
   npm install
   npm run build
   npm start
   ```

## 🛠️ Какие переменные нужны (ОБЯЗАТЕЛЬНО)

Эти ключи нужно скопировать из **Firebase Console -> Project Settings**:

1. `NEXT_PUBLIC_FIREBASE_API_KEY` — API ключ.
2. `NEXT_PUBLIC_FIREBASE_PROJECT_ID` — ID вашего проекта.
3. `NEXT_PUBLIC_FIREBASE_APP_ID` — App ID приложения.
4. `NEXT_PUBLIC_ADMIN_PASSWORD` — Придумайте пароль для входа в `/admin`.
