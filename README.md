# DartBrig Pro - Система управления рейтингами (v2.8 Stable)

Современная платформа для автоматизации дартс-турниров и управления профессиональными профилями игроков.

## 🚀 Готовность к деплою (Any Hosting Ready)
Система полностью автономна и оптимизирована для работы на любых современных облачных платформах:
- **Timeweb Cloud**: Рекомендуется для стабильной работы в РФ. Используйте раздел «Node.js приложения».
- **Vercel / Netlify**: Идеально для автоматической сборки из GitHub.
- **Docker / VPS**: Поддержка Standalone-режима сборки для максимальной производительности.

## ☁️ Подключение Облачной Базы (Firebase)
Облачная база данных подключается к любому хостингу через переменные окружения (Environment Variables). 

**Для Timeweb / Vercel добавьте следующие ключи:**
```env
NEXT_PUBLIC_FIREBASE_API_KEY=ваш_ключ
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=ваш_домен_авторизации
NEXT_PUBLIC_FIREBASE_PROJECT_ID=ваш_id_проекта
NEXT_PUBLIC_FIREBASE_APP_ID=ваш_id_приложения
NEXT_PUBLIC_ADMIN_PASSWORD=ваш_пароль_входа_в_админку
```

*Если переменные не заданы, система запустится в **Demo-режиме** (данные будут храниться в памяти сервера до перезагрузки).*

---

## 🛠️ Особенности настройки на Timeweb
1. **Node.js**: Выбирайте версию 18 или 20.
2. **Команда сборки**: `npm run build`
3. **Команда запуска**: `npm start`
4. **Статический контент**: Проект использует `output: 'standalone'`, что позволяет запускать его на минимальных мощностях Timeweb Cloud.

---

## 📸 Хранение данных
Все данные турниров и фотографии игроков (в формате Base64) сохраняются **внутри облачного контейнера Firebase**. 
1. **Постоянство**: Вам не нужно повторно парсить турниры после деплоя. Данные хранятся вечно до ручного удаления в админке.
2. **Автономия**: Фотографии загружаются в базу и не теряются при перезагрузке сервера или смене хостинга.

Разработано специально для DartBrig. Все визуальные стандарты (10 шаблонов, защита цифр от искажений) встроены в ядро системы.
