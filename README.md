# DartBrig Pro - Система управления рейтингами (v2.7 Stable)

Современная платформа для автоматизации дартс-турниров и управления профессиональными профилями игроков.

## ✨ Новые возможности (v2.7)
- **10 Мастер-шаблонов**: Полная кастомизация карточек игроков (Киберпанк, Элитный, Ретро, Арена и др.).
- **Интеллектуальные подсказки**: Интерактивные справки под знаком «!» с профессиональной аналитикой, адаптированные для мобильных устройств.
- **Advanced Sponsorship**: Поддержка до 3-х спонсоров на игрока с уникальной механикой раскрытия промокодов.
- **Матовое стекло (Deep Blur)**: Обновленный дизайн фиксированных панелей, полностью скрывающий контент при прокрутке.
- **Умные цифры**: Унифицированный размер шрифта (по эталону AVG) с автоматической адаптацией под длину данных.

## 🚀 Как отправить проект в GitHub

Для автоматической загрузки кода в ваш репозиторий запустите команду в терминале Firebase Studio:
```bash
npm run deploy:github
```

## 🛠️ Настройка после деплоя (Для хостинга)

Если вы используете **Timeweb Cloud** или любой другой VPS:

1. **Зайдите на сервер**: Подключитесь через SSH.
2. **Настройте ключи**: Создайте файл `.env.local` в корне проекта.
3. **Добавьте переменные** (скопируйте из Firebase Console -> Project Settings):
   - `NEXT_PUBLIC_FIREBASE_API_KEY`
   - `NEXT_PUBLIC_FIREBASE_PROJECT_ID`
   - `NEXT_PUBLIC_FIREBASE_APP_ID`
   - `NEXT_PUBLIC_ADMIN_PASSWORD` — ваш секретный пароль для входа в `/admin`.

4. **Запуск**:
   ```bash
   npm install
   npm run build
   npm start
   ```
