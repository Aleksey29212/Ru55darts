# DartBrig Pro - Система управления рейтингами (v2.8 Stable)

Современная платформа для автоматизации дартс-турниров.

## 🚀 Как выполнить деплой (GitHub Push) ОДНОЙ КОМАНДОЙ

Я оптимизировал скрипт деплоя. Теперь вам достаточно запустить одну команду:

```bash
npm run deploy:github
```

### ⚠️ Важные нюансы при первой отправке:

1. **Если GitHub выдает ошибку "Push Protection"**:
   В терминале появятся ссылки вида `https://github.com/.../unblock-secret/...`. 
   **Обязательно кликните по ним** и подтвердите разблокировку ("Allow this secret"). Это нужно сделать только ОДИН РАЗ.

2. **Принудительная привязка токена**:
   Скрипт `deploy.sh` теперь сам привязывает токен `ghp_Vd6e...` при каждом запуске. Вам не нужно ничего копировать вручную.
