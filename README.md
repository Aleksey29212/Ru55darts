# DartBrig Pro - Система управления рейтингами (v2.8 Stable)

Современная платформа для автоматизации дартс-турниров и управления профессиональными профилями игроков. Построена на стеке **Next.js 15**, **Tailwind CSS** и **Firebase**.

## 🛠 Технический стек (для специалиста)
- **Framework**: Next.js 15 (App Router)
- **Runtime**: Node.js 18+ / 20+
- **Backend**: Firebase Firestore (NoSQL)
- **Auth**: Firebase Auth (Anonymous)
- **Output**: `standalone` (оптимизировано для Docker и Node-хостингов)

## 🚀 Инструкция по деплою (Timeweb Cloud / Vercel)
Система полностью готова к деплою через GitHub.

**Шаги для установки:**
1. **GitHub**: Загрузите этот код в ваш приватный или публичный репозиторий.
2. **Хостинг**: Создайте проект «Node.js приложение» (на Timeweb) или новый проект на Vercel.
3. **Переменные окружения**: В панели управления хостингом добавьте следующие ключи из вашего проекта Firebase:
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=ваш_ключ
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=ваш_домен
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=ваш_id_проекта
   NEXT_PUBLIC_FIREBASE_APP_ID=ваш_id_приложения
   NEXT_PUBLIC_ADMIN_PASSWORD=ваш_пароль_админа (например, 1234)
   ```
4. **Команды**:
   - Сборка: `npm run build`
   - Запуск: `npm start`

## 📸 Хранение данных
- **Турниры**: Сохраняются в Firestore. Повторный парсинг не требуется.
- **Фотографии**: Загружаются через «Фотостудию», конвертируются в **Base64** и хранятся в документе игрока в Firestore. Это обеспечивает полную автономность — внешние хранилища (S3) не нужны.
