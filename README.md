# DartBrig Pro - Система управления рейтингами (v2.8 Stable)

Современная платформа для автоматизации дартс-турниров и управления профессиональными профилями игроков.

## 🚀 Готовность к деплою (Production Ready)
Система оптимизирована для работы на любом хостинге:
- **Vercel / Netlify**: Идеально для Next.js (автоматическая сборка из GitHub).
- **Firebase App Hosting**: Родная интеграция с Google Cloud.
- **Docker / VPS**: Поддержка Standalone-режима сборки.

## 📦 Основные компоненты
- **Frontend**: Next.js 15 (App Router) + Tailwind CSS.
- **UI**: ShadCN + 10 Мастер-шаблонов личных карточек.
- **Backend**: Firebase Firestore (хранение турниров, профилей и фото).
- **AI**: Google Genkit (интеллектуальная генерация тем оформления).

---

## ⚙️ Настройка перед запуском

Для работы системы в облаке добавьте следующие переменные окружения в панель управления вашего хостинга:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=ваш_ключ
NEXT_PUBLIC_FIREBASE_PROJECT_ID=ваш_id_проекта
NEXT_PUBLIC_FIREBASE_APP_ID=ваш_id_приложения
NEXT_PUBLIC_ADMIN_PASSWORD=ваш_секретный_пароль
```

*Если переменные не заданы, система будет работать в демо-режиме (хранение данных во временной памяти сервера до перезагрузки).*

---

## 📸 Хранение фотографий
Все фотографии, загруженные через **«Фотостудию»**, автоматически конвертируются в формат **Base64** и сохраняются внутри документа игрока в базе данных. Это гарантирует, что фото никогда не потеряются и будут мгновенно отображаться на любом устройстве без настройки внешних хранилищ (S3/Storage).

---

## 🛠️ Команды управления

1. **Загрузка кода в GitHub**:
   ```bash
   npm run deploy:github
   ```
   *Скрипт автоматически исправит ошибки Git и выполнит Force Push в ваш репозиторий.*

2. **Локальная проверка сборки**:
   ```bash
   npm run build
   ```

---
Разработано специально для DartBrig. Все данные (архивы турниров, профили, фото) сохраняются в «контейнере» вашего проекта и не требуют повторного парсинга после первой настройки.
