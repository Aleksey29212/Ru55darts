# **App Name**: DartBrig Pro

## Core Features:

- Tour ID Import: Import darts tournament data by entering tournament IDs from dartsbase.ru, automatically processing and integrating the data.
- Automated Ranking: Automatically calculate and display player rankings based on imported tournament data and a configurable scoring system.
- Configurable Scoring: Enable admins to adjust scoring parameters (base points, bonus points) via a settings panel, influencing ranking calculations.
- Player Card Generation: Generate and display player cards featuring stats, ranking, and a customizable avatar. AI tool is employed to provide useful card styling recommendations, without automatically enforcing those.
- Data Export: Export current ranking data to XLSX format for external analysis or record-keeping.
- Admin Panel: Secure admin login to manage scoring configurations, upload player photos, and edit player data.
- Photo studio: Upload images of players

## Style Guidelines:

- Primary color: Vivid orange (#FF6B00) to reflect the original design, embodying energy and focus, crucial for a competitive dashboard.
- Background color: Dark charcoal (#0A0A0C), a desaturated hue that ensures contrast and legibility while retaining the original dark aesthetic.
- Accent color: A slightly cooler orange (#FF8533) used for highlights and interactive elements, differentiating it from the primary orange for clearer UI cues.
- Body and headline font: 'Montserrat' sans-serif for UI consistency.
- Fallback font: 'Russo One' for emphasis.
- Font Awesome icons for actions and status indicators; clear and consistent visual language.
- Glassmorphism panels with frosted glass effect for depth and focus; subtle layering.
- Subtle transitions for panel appearances; smooth and non-distracting feedback.
