## 🚀 Как выполнить деплой (GitHub Push)

GitHub блокирует отправку, если видит токен в коде. Поэтому мы используем безопасный метод привязки:

1. **Привяжите ваш токен (Один раз)**. Скопируйте и выполните эту команду в терминале:
   ```bash
   git remote set-url origin https://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@github.com/Aleksey29212/Ru55darts.git
   ```

2. **Если GitHub выдает ошибку "Push Protection"**:
   В терминале появятся ссылки вида `https://github.com/.../unblock-secret/...`. 
   **Обязательно кликните по ним** и подтвердите разблокировку ("Allow this secret").

3. **Запустите деплой**:
   ```bash
   npm run deploy:github
   ```
